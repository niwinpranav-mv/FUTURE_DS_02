1 Customer Churn & Retention Dashboard

This project focuses on analyzing customer churn and retention trends using Python, VS Code, and Power BI. The dataset was cleaned and processed using Python and Pandas, then visualized in Power BI through an interactive dashboard.

The analysis identifies key churn drivers such as contract type, payment method, monthly charges, and customer tenure. KPI cards, charts, and slicers were used to create a business-focused dashboard that helps understand customer behavior and improve retention strategies.

2 Tools & Technologies

* Python
* Pandas
* VS Code
* Power BI
* Data Visualization
* Business Analytics

3 Dashboard Features

* Total Customers KPI
* Churn Rate & Retention Rate
* Churn by Contract Type
* Payment Method Analysis
* Customer Tenure Analysis
* Interactive Filters & Slicers

4 Key Insights

* Month-to-month contracts showed the highest churn.
* Customers with higher monthly charges were more likely to churn.
* Long-term customers had better retention rates.
* Electronic check users showed higher churn behavior.

This project demonstrates practical skills in customer analytics, churn analysis, KPI reporting, dashboard development, and business storytelling using real-world business intelligence workflows.
